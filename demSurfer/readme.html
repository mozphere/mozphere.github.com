Basic Instructions

If you are familiar with cmd and batch files, you just need to know the following:
  1. When you select only one replay, you will get a menu.
  2. When you select more than one, the info of all those matches will be written to results.txt
  3. You should run demSurfer by double-clicking the Start shortcut.
  4. You might need to get the latest JRE.

Detailed Info and Insructions

  1. Extract or move the contents of demSurfer.zip file anywhere you like.
    2. Then you will have to either copy/move all three files (Start, cmd.bat, demSurfer.jar) to any directory that contains DotA 2 replays or provide the full path of a replay or a directory that contains replays.
  2. Right click cmd.bat and then click edit.
    3. - You can select one or more replays from the current directory separated by a space 
    java -jar demSurfer.jar example.dem example2.dem
    - or select all replays from the current directory 
    java -jar demSurfer.jar *.dem
    - You can also provide full paths to select replays in different directories 
    java -jar demSurfer.jar "C:\Games\DotA 2 Replays\example.dem"
    - or select all replays in a different directory 
    java -jar demSurfer.jar "C:\Games\DotA 2 Replays\*.dem"
  3. Now you can double-click Start. You will find match info and scoreboards for all the replays in either example.txt or results.txt

Files

  • Start.lnk is just a shortcut to the Windows Command Prompt that also changes the console's font to one that does support Unicode characters.

  • cmd.bat is a batch file that executes the commands written in it.
    • chcp 65001 changes the Code Page so that Unicode (UTF-8) characters are supported.
    • cls just clears the screen.
    • java -jar demSurfer.jar *.dem runs the program, selecting all files in the current directory.
  • demSurfer.jar is an archive file that contains all the necessary files to run the program.
